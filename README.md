# sls-common-tools

These are all common utilities

## Installation

```sh
npm install sls-common-tools --save
```

## Usage

### TypeScript

```typescript
import { log } from 'sls-common-tools';
log.debug('debug test');
log.info('info test');
```

```sh
Output should be 'Geese'
Output should be 'Gun'
```

## Test

```sh
npm run test
```
