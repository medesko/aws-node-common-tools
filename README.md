# sls-common-tools

These are all common utilities

## Installation

```sh
npm install sls-common-tools --save
```

## Usage

### TypeScript

```typescript
import { log } from 'sls-common-tools';
log.info('info test');
```

```sh
Output should be {"level":30,"time":1575329413411,"pid":80837,"hostname":"MBP-DE-MOHAMED","msg":"info test","v":1}
```

## Test

```sh
npm run test
```
